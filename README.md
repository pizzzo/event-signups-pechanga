# Event Signups – SPA Demo

## Overview
This project is a small **SPA-style webpage** built to demonstrate **core JavaScript fundamentals** in a clean, framework-agnostic way. My goal is show my current progress in understanding web-dev core concepts.

## Tech Choices
- **Vanilla JavaScript** – Focus on fundamentals without framework abstraction.
- **SPA-style architecture** – client-side interactions without full page reloads.

## Prerequisites
- Node.js (recommended: latest LTS)
- npm (comes with Node.js)

## Installation & Launch Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/pizzzo/event-signups-pechanga.git

2. Enter the project directory:
   ```bash
   cd event-signups-pechanga

3. Install dependencies:
   ```bash
   npm install

4. Run development server:
   ```bash
   npm run dev

5. Open the application in your browser at:
   ```browser
   http://localhost:5173


## Purpose
This project is intentionally kept small and simple to:
- Highlight core JavaScript\HTML\CSS concepts
- Avoid framework lock-in. 
- Serve as a foundation for future expansion or framework adoption.
